# График Контрибьюции

## Для Портфолиов

Этот проект разработан как портфолио для демонстрации навыков. Если вы желаете использовать его в качестве базы:

1. **Сделайте Fork** репозитория
2. **Удалите** ссылки на этот проект
3. **Замените** на свое доменное имя
4. **Адаптируйте** тесты к вашей бизнес-логике
5. **Опубликуйте** свою версию

## Стандарты Кода

- **Python**: 3.11+
- **Type Hints**: Обязательные для всех функций
- **Тестирование**: Всему новому коду нужны тесты
- **Покрытие**: Остановить на 85%+
- **Форматирование**: Black (длина строк: 100)
- **Linting**: flake8
- **Импорты**: isort

## Настройка Pre-commit

```bash
pip install pre-commit
pre-commit install
```

## Требования к Тестам

- Новые фичи должны включать unit + integration тесты
- Все тесты должны проводиться локально
- Тесты должны иметь определённые маркеры
- Нет скипам тестов без уважительных причин

## Процесс Pull Request

1. Новая ветка: `git checkout -b feature/my-feature`
2. Осуществите тесты
3. Прогоните полные тесты: `pytest -v`
4. Проверьте покрытие: `pytest --cov=app`
5. Комит: `git commit -m "Add: clear description"`
6. Push и создасте PR
